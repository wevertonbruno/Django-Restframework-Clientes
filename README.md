# Django-Restframework-Clientes

<h3>Endpoints:</h3>

<ul>
  <li>GET
    <p>url_base/api/clientes</p>
    <p>url_base/api/clientes/:name:/</p>

<li>PATCH

<li>DELETE
  <p><p>url_base/api/clientes/:name:/delete</p></li>
<li>POST
  <p>url_base/api/clientes/create</p></li>
